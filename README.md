## Getting Started
```
$ git clone https://github.com/GitMullins/personal-bio-site
```
### Prerequisites
```
$ npm install -g http-server
```
## Running
Browse to personal-bio-site directory and run http server  
```
$ hs -p 5000
```

in your web browser navigate to `localhost: 5000`
